# RoleRadar Portfolio Story

## What this project is
RoleRadar is an end-to-end analytics product that converts raw job postings into explainable opportunity scores. It combines text feature engineering, supervised machine learning, batch scoring, an API layer, and a lightweight dashboard.

## The problem
Most job-search projects stop at salary prediction. That is easy to understand, but narrow. RoleRadar reframes the problem around **decision support**:
- Which postings look strongest for applicants?
- Which skills show up most often?
- Which roles are accessible for early-career candidates?
- Which postings look current, flexible, and business-relevant?

## What makes this more employable
This repo demonstrates:
- data ingestion and preprocessing
- NLP-style signal extraction from job descriptions
- structured feature engineering
- machine learning model training and evaluation
- API deployment patterns
- dashboarding and product-facing presentation
- test coverage on critical feature logic

## How to explain the score in an interview
The opportunity score is not a claim that a job is objectively better than another in every context. It is a ranking signal derived from:
- technical stack depth
- modern tooling references
- flexibility signals
- barrier-to-entry signals
- company context
- posting quality and business exposure

That framing makes it useful for prioritization and market analysis, not just prediction.
